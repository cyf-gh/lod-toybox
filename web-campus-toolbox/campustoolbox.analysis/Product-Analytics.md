# 产品策划书

## 项目背景与市场

高校学生主要通过微信和QQ社交，缺少更加直观的校内同平台交流软件。

### 事例选取

1. 学生间的二手闲置物品交换依靠创建微信群和咸鱼等面向全社会类APP，微信群效率不高，曝光率太低，而且学生换届后要重新创建，咸鱼等APP则直面社会，虽然拥有很高的曝光度却缺少了便捷性和安全快速的优势。
2. 叫外卖或帮忙跑腿拿东西在高校极为普遍，同样只能依靠微信群，效率低且缺乏时效性（比如很急的事情需要帮拿东西，刚好群内的人没有有空的）。
3. 学生间社交的途径太少，缺乏泛型开阔交流方式，使得学生间没法形成一个覆盖大多数学生的交流平台。这里面会出现很多需求，比如喜爱学习的同学苦于找不到一起学习的同伴，喜爱打游戏的同学找不到一起开黑打游戏的同伴，想要找男女朋友却没有有效的方式扩大交流环境（大学很多社交都是依靠酒吧夜店，这并不适用于所有人）
4. 学校的老师和学生间同样联系极其局限，大学生具备一定的专业能力，高校老师经常会有一些需要同学帮助的事情，而老师通常都能支付酬劳给想要赚钱的学生，光依靠微信群不仅没有效率，而且杜绝了广大没有加群的学生参加的可能性。
5. 各专业之间很难大范围交流，杜绝了学术之间的碰撞，各学院只有十分局限的交流，而普通学生可能没有交流的机会，（例如服装学院想要拍照找模特，如果你不认识模特专业的学生就会很麻烦，而材料专业可能有什么实验需要和化学专业的同学交流，同样苦于没有联系方式）
6. 学校的社团并不能融入大多数普通学生之间，开学招新只能依靠跑腿发传单，如果不关注各社团的公众号甚至无法对其有了解，信息的传递低效且被动。（新生可能不知道这个社团到底是干什么的，而如果没有人去你们宿舍发传单，你甚至不知道有这个社团这个极大的违反了社团发展的初衷）
7. 学生会等官方机构缺少一个公共的平台来增加存在感，甚至很多学生和到毕业都不会和这些团体产生交集。
8. 校外企业或者个人需要发布兼职只能靠兼职群打广告，而这个方式低级且很局限，没加群就看不到，同样，别人发的时候没看到，过段时间就被刷掉，不往上找就看不到。
9. 外校的学生间如果离得不近很难有交流，甚至本校如果不在一个校区也不会有什么交集，而依赖微信的加好友或者微信群则很被动低效（我可能要问很多人才能找到一个我想要加的学校的学生，而如果有APP在对方学校的页面直接行动就能方便很多）
10. 总之各种各样的问题在如今只依靠微信的情况下不能方便的解决，由于微信的添加好友这个限制和群功能的局限（群更适合即时聊天，而不能依靠他作为一个平台发布信息，定位不同）一个免费方便快捷的校内专门的平台是很必要的。

 

APP产品/服务

APP将主要通过校内身份绑定（通过学号等绑定真实身份）作为身份依据，主要通过类似于任务大厅发布酬劳任务（如A在某宿舍需要同学帮带早餐，酬劳为5元/某老师需要某同学帮忙等酬劳为100元/某企业需要同学参加某活动酬劳200元等）将以标题形式放在页面上，可以自行通过消耗金钱等等方式来增加曝光率，点击页面链接后将转到详细内容，下方则设置评论区供参考，交易则设置类似交易市场的平台，作为内置模块，风格类似于线上销售app，通过图文描述等，部分物品则可以线下面议价格。app主页将以各学校新闻为主，如某专业将进行实验展出，某校发生某事，等等来增加用户黏度，可以考虑聘请专门的学生作为编辑（学校如同小型社会）。App同样设置“发现“页面，将不同于朋友圈的好友可见，作为同校学生，在部分隐私性问题上可以得到更好的解决方式，页面将以类似网游的全用户在线形式，让所有人在同一块页面中发言，通过专业/年级/性别等等要素来做大体区分细化，宏观上保持多人群聊模式。App与微信设置用户名授权，如果想要加好友可直接导向微信，这样既避免与微信等大企业直接争夺社交市场，同样也在一定程度上与微信相辅相成，更加突出产品做**多用户无局限同平台交流**的特点。

 

### 目标市场分析与竞争优势

大学市场极为广阔，而同样做大学APP的很少，目前比较有名的是“易班app”，而易班的用户评价极为差，完全无好评，通过对易班的分析，得出的结论为

1. 易班采取强制注册的行为，造成用户体验极差
2.  易班本身没有任何特色，全盘照抄微信的社交体系
3.  易班在主页设计上杂乱无章，产品定位模糊，交互界面差
4.  推送新闻与多数学生生活无关，过于架空
5.  无法拥有稳定的流量导致恶性循环（根本用不到）

#### 对比易班APP，得出优势为

1. 将采取柔性宣传，减少注册所产生的用户使用成本（如利用学生的学号注册）
2. 将突出APP作为大平台交流APP的特色，不与微信争夺市场，
3. 主页上以更加贴近学生生活的新闻为主，以更加简洁明快的交互增加用户体验
4. 拥有很多切实提高用户生活质量的功能，将提高用户的主动性
5.  以用户自身的生活环境为背景，提高APP的接受程度

### 目标市场和需求

#### 消费人群和定位

各大高校的在校生

#### 竞争优势

切实解决生活上的问题，学号注册减少注册成本，贴近生活，更加有趣的交流方式，学生间人脉的温床  将拥有很多的发展空间

#### 盈利模式

初期以流量为主，投放适量广告，其余服务不收取中间费，对于校外需求可适当收费

 

#### 知识产权

运营模式的创新和社交模式的创新

 

#### 战略和计划

通过在各大微信群贴吧投放广告，人工宣传，与社团合作，一个宿舍一个宿舍的跑等方式一步步扩大用户量，初具规模后可适当与学校官方接触，争取得到官方认证。计划能够覆盖全校，之后则将上海其他高校作为目标，加大宣传力度，争取能够在上海的高校推广并运营。

 

 

#### Swot分析

作为在校生拥有更加直观的体验，能够直面用户需求

作为学生缺乏经济支撑和技术支持

市场上做校内社交软件的很少（几乎没有），市场前景较好

盗版的入侵门栏低，与大公司相比缺少竞争力
